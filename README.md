# GTM Lead Qualification Automation

This Python project automates lead qualification and prioritization using CSV input and output workflows.

## Features

- Reads lead data from a CSV file
- Classifies leads based on score and company size
- Assigns lead priority levels
- Exports processed leads into a new CSV file
- Tracks qualification summary counts

## Technologies Used

- Python
- CSV file handling
- Functions
- Loops
- Conditional logic

## Input

The script reads lead data from:

```
leads.csv
```

## Output

The script exports processed leads to:

```
processed_leads.csv
```

Example Workflow:

CSV Input
> Python Processing
> Lead Classification
> Priority Assignment
> CSV Export

## How to Run:
Run the script with
python lead_qualifier.py
